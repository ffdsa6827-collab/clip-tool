# Clip Tool — موقع مفتوح المصدر لقص الفيديو في المتصفح

هذا مشروع بسيط يسمح لك بقص مقاطع من فيديو محلي مباشرة في المتصفح باستخدام ffmpeg.wasm. لا يحاول المشروع تجاوز ميزات مدفوعة في خدمات أخرى.

ميزات:
- رفع ملف فيديو محلي
- تحديد وقت البداية والنهاية بالثواني
- قص المقطع باستخدام ffmpeg.wasm
- تنزيل المقطع الناتج

متطلبات:
- متصفح حديث (Chrome, Edge, Firefox حديثة)
- اتصال إنترنت لتحميل مكتبات ffmpeg.wasm في التشغيل الأول

كيفية الاستخدام محلياً:
1. فك ضغط الملفات أو انسخها لمجلد.
2. افتح `index.html` في المتصفح مباشرة (قد تحتاج خادم محلي لبعض المتصفحات).
   - لتشغيل خادم بسيط: `python -m http.server 8000` ثم افتح http://localhost:8000

نشر على GitHub Pages (الطريقة الأسهل):
1. سجّل دخولك في GitHub.
2. أنشئ مستودعًا جديدًا باسم تريده، مثلاً `clip-tool`.
3. ارفع الملفات (`index.html`, `styles.css`, `script.js`, `README.md`) إلى الفرع `main`.
4. اذهب إلى Settings > Pages، واختر Source = main branch / root ثم احفظ.
5. بعد بضع دقائق سيكون موقعك متاحاً على: `https://<your-username>.github.io/<repo-name>/`

نشر على Vercel:
1. أنشئ مستودع GitHub وادفع الملفات إليه.
2. سجل دخول إلى Vercel وربط حساب GitHub.
3. استورد المشروع وجرّب النشر — Vercel سيعطيك رابطاً مباشراً.

ملاحظات مهمة:
- لا أستطيع ولا أدعم أي طريقة للحصول على ميزات مدفوعة في خدمات تجارية بشكل مجاني غير قانوني.
- إذا أردت خواص متقدمة (تحويل صوتي/نص إلى كلام، تحويل تلقائي للترجمة من الصوت، قص تلقائي ذكي)، أقدر أريك كيف تدمج مكتبات مفتوحة المصدر (مثل whisper.cpp أو نماذج مفتوحة أخرى) أو واجهات API مجانية/رخيصة — لكن بعض الوظائف المعقّدة تحتاج قدرات حسابية كبيره أو خدمات سحابية مدفوعة.
